What is OntoUML?

OntoUML is a UML Profile whose elements represent finer-grained distinctions between different kinds of types and encompasses a set of constraints defining the admissible relations between these elements.

What is the OntoUML SDK?

The OntoUML SDK is composed of two components:

  • PrimeOntoUML - the (Ecore) reference metamodel for OntoUML.
    It's a lightweight version of Roberto Carraretto's version (which is based on Eclipse's UML metamodel).
  • OntoUML Editor - a textual editor for OntoUML.
    Available as an Eclipse plugin, the editor integrates with several modelling tools from the Eclipse Modelling project.

Installation

The Eclipse plugins can be obtained from the OntoUML SDK update site, available at:

http://ontouml.sourceforge.net/eclipse/

The plugins are intended to be used in Eclipse Juno release.
